# 📝 Python Task Manager

A simple command-line **Task Manager (To-Do List System)** built in Python.  
No external libraries required — only core Python modules.  

---

## 🚀 Features
- ✅ Add tasks  
- ✅ View tasks  
- ✅ Delete tasks  
- ✅ Auto-save tasks to a file (`tasks.txt`)  
- ✅ Loads saved tasks when restarted
